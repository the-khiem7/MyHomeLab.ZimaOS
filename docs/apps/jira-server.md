# Jira Server

Self-hosted Jira Server for project management.

## Configuration

Source File: [JiraServer.dockercompose.yml](../../app/JiraServer.dockercompose.yml)

## Deployment

```bash
docker-compose -f app/JiraServer.dockercompose.yml up -d
```

> **Note**: Jira Server requires significant memory resources. Ensure your device has at least 4GB RAM available.
